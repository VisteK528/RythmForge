Introduction
============

The goal of the project was to implement a library with functionality similar to the (`librosa <https://librosa.org/>`__) library, but with a backend in C++.

Tested on platforms:

- `Ubuntu 22.04 LTS`
- `macOS Sonoma 14.3`
- `Windows`

Used language versions:

- Python: `3.10.x`
- C++: `20`
